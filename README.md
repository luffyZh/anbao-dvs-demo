# 《H安保》隐私防护核安保 Demo（单页大屏）

这是一个**尽量简单**的交互 Demo：单页大屏 + 本地轻量服务，用于对客讲解“事件相机人员经过 → 台账对齐 → 不合规告警 → 脱敏复核”的闭环。

## 1. 启动方式
在本目录执行（Node.js 后端，推荐）：
```bash
npm install
npm run dev
```
然后浏览器打开：
```
http://localhost:8000/
```

（备选：Python 标准库版，零依赖）
```bash
python3 app.py
```
然后浏览器打开：
```
http://localhost:8000/
```

## 2. 你能演示什么
- 左侧：实时事件流（最新在上）
- 右侧：复核详情（默认展示占位 mp4；新事件会在两个 mp4 间轮换）
- 顶部：今日事件/告警/未处理统计
- 交互：
  - “模拟新事件”：写入一条新的事件 JSON，页面会自动刷新
  - “确认违规/误报/置为未处理”：改变事件状态（写回 JSON）

## 3. 事件数据（文件落盘）
事件 JSON 位于：
```
events/YYYY-MM-DD/*.json
```
静态素材位于：
```
static/assets/
```

你后续给我更多真实 DVS 图片/视频后，我们可以：
1) 直接替换 `static/assets/placeholder.gif` / `placeholder.mp4`；或  
2) 改成“每条事件对应自己的素材文件名”，实现更真实的复核体验。
